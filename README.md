## 📌 Overview

**CacheProxyComputer** is a lightweight proxy application developed in **Go**, focused on delivering fast and reliable internet proxy functionality with minimal system overhead.

The project is designed for developers, network enthusiasts and advanced users who need a simple yet efficient proxy solution that can run smoothly on Windows environments while maintaining portability for future cross-platform support.

Its architecture prioritizes:

* Performance
* Low memory usage
* Simplicity
* Scalability
* Maintainability

## 🧠 Built With

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/MGNG13/CacheProxyComputer.git
```

### Enter the project directory

```bash
cd CacheProxyComputer
```

### Run the project

```bash
go run .
```

### Build executable

```bash
go build -o CacheProxyComputer.exe
```
